springboot集成logback笔记

发表于 2021-02-22 | 分类于 Spring

springboot 集成 logback 的学习笔记。

笔记都在以下配置的注释中。在示例配置中，只是简单做了 dev 和 prod 环境的示例配置。其中 dev 环境，全部输出到控制台，且 mybatis 日志使用 p6spy 记录。prod 环境则全部输出到文件。

配置实例

springboot 配置

immediateFlush：logback 使用缓存先记录日志，等缓存满后在刷新到文件。由于 logback 使用 appender 记录日志，如果 appender 未正确关闭而退出，会损失一部分日志，大小约为 8k。如果要求日志安全性，则设置为 true 立即刷新，不会丢失日志。但设置为 false，可以获取更大的吞吐量。

logback-spring.xml 配置

logback-spring.xml 以 spring 结尾，可以使用 springboot 一些自定义的标签。

spy.properties 配置

这里，关闭 mybatis 自带的日志，使用 p6spy 日志框架代替。可以有更好的输出格式，没有 ‘?’ 占位符，还能输出执行时间。配置文件直接放在 resources 目录下即可。
